About Us

This is a lightweight public API made for entertainment purpose. The API is still in development stage so more quotes will be added in future. To report a bug or suggest an enhancement please contact us


  • Endpoints

  • Get random quotes

    https://friends-quotes-alpha-api.herokuapp.com/friends/random/

  • Get random quotes by character

    https://friends-quotes-alpha-api.herokuapp.com/friends/random/character/Joey

  • Get random quotes by level

    https://friends-quotes-alpha-api.herokuapp.com/friends/random/level/Easy

  • Get random quotes by Season

    https://friends-quotes-alpha-api.herokuapp.com/friends/random/season/S01

  • Get random quotes by Character and Season

    https://friends-quotes-alpha-api.herokuapp.com/friends/random/season/character/S01/Joey

  • Get list of all Characters by Season

    https://friends-quotes-alpha-api.herokuapp.com/friends/characters/S01

Allowed HTTP methods

Currently the API only supports GET requests.


Status Code

The API will respond with one of the following status codes

Code Description
200 Success
400 Bad Request
401 Unauthorized
403 Forbidden
404 Not Found
500 Internal Server Error

Rate Limit

The current version have rate limit of 10 calls per hour with maximum of 50 calls per day
